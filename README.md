# SAGE: Scientific Analysis of Graph-based Epistemes

Aim: characterize scientific paradigms during the chemical revolution

Objectives:
- Classification of scholarly documents
  - Evaluate if LDA-topics reflect concepts
  - Evaluate if LDA-topics dynamics map onto paradigm shifts
- Identification of contextual factors driving revolutionary change
  - Find nodes leading the development of each concept's subgraph
  - Find nodes/cliques defining differentiation among subgraphs
  - Find nodes increasing "connectivity" in the general network

Methods:
- Classification -> LDA, NME
- Identification -> KnowFormer
